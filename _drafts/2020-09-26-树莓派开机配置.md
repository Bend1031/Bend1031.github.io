若是没有屏幕，利用无线网，则要开启 ssh，与配置 wifi。
1. boot 分区中新建 ssh。
2. 新建 wpa_supplicant.conf 文件，
```
country=CN
ctrl_interface=DIR=/var/run/wpa_supplicant GROUP=netdev
update_config=1
network={
 ssid="wi-fi name here"
 psk="wi-fi password here"
 key_mgmt=WPA-PSK 
 priority=1
}   

**注意如果连接 5G wifi 可能存在信道干扰，建议先用 2.4G wifi 连接**
